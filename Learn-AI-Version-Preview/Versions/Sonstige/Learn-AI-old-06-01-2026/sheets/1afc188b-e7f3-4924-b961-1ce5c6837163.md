# Physik-Arbeitsblatt: Gleichförmige Bewegung   

**Aufgaben:**   
  
1. Bestimmen Sie die Geschwindigkeit eines Fahrradfahrers, der eine Strecke von 500 Metern in 100 Sekunden zurücklegt. Beachten Sie die Formel für die Geschwindigkeit.   
  
2. Ein Auto bewegt sich mit einer konstanten Geschwindigkeit von 72 km/h. Wie weit ist es in 15 Minuten gefahren? Denken Sie daran, die Zeit in Stunden umzurechnen.   
  
3. Ein Zug legt eine Strecke von 200 Kilometern in 2 Stunden zurück. Berechnen Sie seine Geschwindigkeit in km/h.   
  
4. Ein Sportwagen fährt mit einer Geschwindigkeit von 30 m/s. Wie lange braucht er, um eine Strecke von 1.8 Kilometern zurückzulegen? Vergessen Sie nicht, die Strecke in Meter umzurechnen.   
  
5. Ein Flugzeug fliegt mit einer Geschwindigkeit von 900 km/h. Wie weit ist es in 45 Minuten geflogen? Denken Sie daran, die Zeit in Stunden umzurechnen.   
  
**Tipps:**   
- Nutzen Sie die Formeln für Geschwindigkeit, Zeit und Strecke.   
- Denken Sie daran, Einheiten umzurechnen, wenn nötig.   
- Überprüfen Sie Ihre Antworten auf Realistisches.
