# Learn-AI Version Lists

Diese Flask-WebApp liest alle Unterordner aus `Versions` ein und zeigt pro Version eine Vorschau an.

## Start

```powershell
python -m venv .venv
.venv\\Scripts\\Activate.ps1
pip install -r requirements.txt
python app.py
```

Danach im Browser `http://127.0.0.1:5000` oeffnen.

## Nutzung

- Ganze Projektordner nach `Versions` kopieren
- Seite neu laden
- Links eine Version auswaehlen
- Rechts README, Struktur, wichtige Dateien und Bilder ansehen
