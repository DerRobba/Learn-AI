# Learn-AI Model Compare

Vergleicht ein Basismodell direkt mit einem feingetunten Modell in einer gemeinsamen Oberfläche.

## Start

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

Danach im Browser `http://127.0.0.1:5000` öffnen und die `.env` mit deinen Endpoints, Modellnamen und API Keys befüllen.
